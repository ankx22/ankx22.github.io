Projects
Navigating the Unknown- Optical flow based structure-less gap detection for drone flight
project page

In this project, we develop an advanced perception framework for the DJI Tello EDU drone, aimed at enabling autonomous navigation through windows of irregular and unknown shapes. The core objective is to accurately identify and traverse the largest available aperture within a barrier.

The procedure initiates by positioning the drone to ensure a comprehensive view of the entire opening, followed by capturing the window's optical flow. This flow information is then refined through post-processing techniques to delineate the contour of the largest opening and locate its central point. Once the center is determined, visual servoing techniques are utilized to align the drone's camera center with the opening's midpoint, enabling a precise and successful navigation through the opening. The Perception Saga- Sim2Real Quadcopter Perception Stack
project page

In this initiative, a sophisticated perception system was engineered for the DJI Tello EDU drone, designed to facilitate accurate maneuvering through various windows with pre-known textures and approximate sizes.

To realize this objective, simulation-to-reality (sim2real) strategies, including domain randomization, were employed to create a varied synthetic dataset using Blender. This dataset was then utilized to train a segmentation network capable of identifying the foremost window. Upon determining the window's center and corners, the Perspective-n-Point (PnP) technique was applied to establish 2D-3D correspondences essential for precise navigation.

Fly Through Real Trees- Quadcopter motion planning stack
project page

In this project, a comprehensive path and motion planning framework, integrating RRT* (Rapidly-exploring Random Trees Star) for path planning and minimum snap trajectory generation for motion planning, is deployed for 3D navigation of a DJI Tello Edu drone. This framework is initially tested in a simulated environment using Blender and subsequently applied to the real quadcopter, controlled via an NVIDIA Jetson Orin Nano.

Furthermore, cascaded PID (Proportional-Integral-Derivative) controller gains dedicated to position and velocity control are meticulously tuned to ensure adherence to the generated trajectory. (hover mouse on gif to change from real to sim)

3D attitude estimation using various filters
project page

The project involved estimating the 3D orientation or attitude from 6 Degrees Of Freedom (DOF) IMU data, which includes inputs from an accelerometer and gyroscope. This estimation was performed using five distinct methods: employing solely accelerometer data, using only gyroscope data, applying a complementary filter, utilizing a Madgwick filter, and implementing an Unscented Kalman Filter (UKF).

Multi-Robot Motion Planning for Warehouse Management
project page

Developed a system for autonomous robot navigation in warehouses, using Cooperative A* for efficient, collision-free operation. Enhanced navigation through algorithms for real-time obstacle adjustment, improving safety and efficiency. Tested methods, particularly WHCA*, showed superior dynamic obstacle management and multi-robot coordination, promising to revolutionize warehouse logistics.

3D RRT Planner for gear shaft removal

Developed an advanced Rapidly-exploring Random Tree (RRT) motion planning system designed to extract the mainshaft from a gearbox's transmission case without causing any collisions, ensuring damage prevention. This innovative approach enhances precision and safety during the removal process, significantly mitigating the risk of damage to critical gearbox components.

Robust Trajectory Tracking for Quadrotor with Sliding Mode Control
project page

Developed a compact sliding mode control for the Crazyflie 2.0 drone, enabling precise control over altitude and attitude. This controller ensures accurate navigation along desired paths and waypoints, effectively resisting external disturbances for stable flight dynamics.
